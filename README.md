### Required Libraries
```
SDL2 (SDL.h)
SDL2_image (SDL_image.h)
SDL2_ttf (SDL_ttf.h)
```

### Additional Linker Dependencies
```
SDL2.lib
SDL2main.lib
SDL2_image.lib
SDL2_ttf.libg
```

### Required DLL Files
Place these in the root folder, where the include and source folders are:
```
libfreetype-6.dll
libpng16-16.dll
SDL2.dll
SDL2_image.dll
SDL2_ttf.dll
zlib1.dll
```
